# 🔍 Smart and Secure Lost Item Recovery System

An AI-powered web application that helps students and staff report, search, and recover lost items within a college campus.

## 🚀 Features

* User Registration & Login
* Report Lost Items
* Report Found Items
* AI-Based Match Detection
* Email Notifications
* Ownership Claim System
* Admin Dashboard
* Analytics & Reports

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

### Backend

* Python Flask

### Database

* MySQL

### AI/ML

* Sentence Transformers (`all-MiniLM-L6-v2`)
* Scikit-learn
* Cosine Similarity

## 🤖 How It Works

1. User reports a lost or found item.
2. Data is stored in MySQL.
3. AI compares item descriptions.
4. Similarity score is generated.
5. If a match is found, email notifications are sent.
6. User submits a claim for verification.

## 📌 Project Flow

Register → Login → Report Item → AI Matching → Notification → Claim Request → Admin Verification → Item Recovery

## Project objectives 


Today, I focused on defining the project objectives for the Smart and Secure Lost Item Recovery System. The primary goal is to develop a secure and user-friendly platform that helps students and staff report lost and found items efficiently. Key objectives include enabling easy item reporting, implementing AI-based matching to identify potential item matches, sending automated notifications to users, and providing a claim verification process for secure item recovery. Additionally, the system aims to improve recovery rates, reduce manual effort, maintain a centralized database, and ensure data security through proper authentication and access control mechanisms.

## 🔒 Security

* Password Hashing
* Email Verification
* Role-Based Access Control
* Input Validation

## 🎯 Goal

To provide a smart, secure, and efficient platform for recovering lost items using Artificial Intelligence.
